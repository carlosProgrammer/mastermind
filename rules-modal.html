Game rules
Introduction

Master Mind is an abstract board game of cryptanalysis. The player must guess the secret code composed by the server.

The secret code is a series of 4 colors (that can be repeated), automatically generated every new game.

Game stages

The player makes his attempt, trying to guess the code. The server responds by providing some suggestions:

  • The number of right colors in the right place, marked with black circles

  • The number of right colors in the wrong place, marked with white circles

Victory

If the decoder (player) succed to guess the code within 10 attempts, he wins the game, otherwise the encoder (server) wins.
